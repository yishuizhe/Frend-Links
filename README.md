# 友情链接页面

这是一个简单的友情链接页面，用于展示一组友情链接。页面包括友情链接列表以及分页器，方便用户浏览大量链接。

## 功能

- 显示友情链接列表
- 支持分页浏览

## 如何使用

1. 将 HTML 文件下载到本地。
2. 双击 HTML 文件在浏览器中打开，或者将文件部署到 Web 服务器上进行访问。
